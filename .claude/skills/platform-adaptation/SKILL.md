## Content Adaptation Workflow

### Step 1: Analyze Source Content

Identify:
- Core message/value
- Target audience
- Content type (tutorial, story, opinion)
- Key takeaways
- Emotional tone

### Step 2: Select Target Platforms

Choose platforms based on:
- Content fit with platform culture
- Target audience demographics
- Content format compatibility
- Resource constraints

### Step 3: Adapt for Each Platform

**For each platform:**
1. Apply platform-specific style
2. Optimize length for platform
3. Add platform-specific formatting
4. Include platform-specific CTAs
5. Use platform-optimized hashtags/tags

### Step 4: Quality Check

Verify:
- Platform requirements met
- Algorithm optimizations applied
- Brand voice maintained
- No platform-specific violations
